一

Сравнительные конструкции

Напиши по одному предложению с каждой структурой. Нажми кнопку ✨ для проверки с ИИ.
比 A比B + прил. — A [прил.]-ее, чем B
更 A更 + прил. — A ещё более [прил.]
没有 A没有B + прил. — A не такое, как B
最 A最 + прил. — A самое [прил.]
📖 Образец — читай, слушай и запоминай структуру
比 A дешевле B
苹果比梨便宜。 Píngguǒ bǐ lí piányí. Яблоко дешевле, чем груша.
更 A ещё дешевле
苹果更便宜。 Píngguǒ gèng piányí. Яблоко ещё дешевле (более дешёвое).
没有 A не такое дешёвое, как B
苹果没有梨便宜。 Píngguǒ méiyǒu lí piányí. Яблоко не такое дешёвое, как груша.
最 самое дешёвое
梨最便宜。 Lí zuì piányí. Груша самая дешёвая.
二

很 или 是? Переведи на китайский

Переведи русские предложения на китайский язык, соблюдая грамматическую форму. Нажми ✨ для интеллектуального разбора.
很 Прилаг. = сказуемое (Этот библиотека тихая) → 这个图书馆很安静。
是 Прилаг. = определение (Это тихая библиотека) → 这是一个安静的图书馆。
📖 Образец — обрати внимание на разницу структур
很 прил. = сказуемое
Эта библиотека тихая. 这个图书馆很安静。 Zhège túshūguǎn hěn ānjìng.
是 прил. = определение
Это тихая библиотека. 这是一个安静的图书馆。 Zhè shì yī gè ānjìng de túshūguǎn.
